# HTML-CSS-Project
This landing page is designed to convert visitors into students by highlighting key course features, social proof, and a clear call to action. It includes a hero section with student registration counts, a pricing table with a countdown timer, and a mobile-responsive navigation bar.

Key Features
Hero Section: Highlights 100+ hours of live classes and industry-level projects.

Trust Signals: Displays logos of top companies like Amazon, LinkedIn, and Uber that utilize Java Spring.

Pricing & Urgency: Features a countdown timer and a "15% OFF" discount badge with a coupon application field.

Responsive Design: Fully optimized for mobile devices using CSS media queries, including a hamburger menu.

Certificate Preview: Showcases the value of completion through official and verified certificates.

FAQ Section: Provides quick answers to common student queries regarding live sessions and certification.

🛠️ Built With
HTML5: For structured, semantic web content.

CSS3: Custom styling using Flexbox and Grid for complex layouts.

Google Fonts: "Poppins" for a clean and professional aesthetic.

📂 File Structure
index.html: The main structure of the landing page, including all sections from navigation to footer.

styles.css: The stylesheet containing all layout rules, color variables, and responsive breakpoints.

/image: Directory for all graphical assets, including logos and icons.
